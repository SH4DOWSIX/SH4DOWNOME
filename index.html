What it does

Tools for structured rhythm practice.

Routine builder

Create a piece, add labelled sections for the parts you are working on, reorder them, rename them, or bulk-create tempo ramps.

Subdivision library

Pick standard, compound, mixed, and tuplet patterns including triplets, quintuplets, septuplets, dotted values, and rests.

Custom pattern editor

Build reusable subdivisions with notes, rests, accents, dotted notes, and custom pulse groupings.

Polyrhythm mode

Set ratios like 3:2, 5:4, or 7:4, choose per-beat or per-bar scope, and see the relationship while it plays.

Practice automation

Use tap tempo, count-in, countdown timer, and speed trainer with bars-per-step, BPM step, and maximum tempo controls.

Visual feedback

Follow beat circles, subdivision pulses, polyrhythm grids, timer remaining, and detachable Beat Window styles.

Setup

Keep your practice setup portable.

SH4DOWNOME saves pieces, custom subdivisions, sound choices, visual preferences, and Beat Window settings between sessions.

1 Backup & restore Export selected pieces and custom subdivisions, then import them later.
2 Personal settings Choose sound set, accent colour, terminology, and Beat Window auto-open.
3 Desktop extras Use always-on-top and keyboard shortcuts for faster section control.
