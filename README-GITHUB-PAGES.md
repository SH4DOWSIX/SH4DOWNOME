## How to publish

Copy all files/folders into the root of your `SH4DOWNOME` GitHub repo.

Then:

1. Go to the repo on GitHub.
2. Open **Settings**.
3. Open **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Branch: **main**
6. Folder: **/ root**
7. Save.

GitHub should publish it here:

`https://sh4dowsix.github.io/SH4DOWNOME/`
